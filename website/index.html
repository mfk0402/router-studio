Built for real work

Router Studio is an Electron desktop app: your project opens in a folder you choose, and your OpenRouter key stays on device. No cloud account is required for the core product. An optional Router Studio account (local vault) syncs encrypted settings between sessions on the same machine — see Account. Highlights below; the full map is under Capabilities.

OpenRouter & Free Mode

Use any model available through OpenRouter, or enable Free Mode with the free router or round-robin over discovered free models. Optional fallback model if a free tier is rate-limited.

Model marketplace

Category sidebar (coding, vision, reasoning, free, large context, …), price tiers Free/$/$$/$$$, sort by price or context, and quick picks per category.

Monaco & tabs

60+ languages, tabs, split editor, Format Document, Format on Save, Quick Open, and diff preview with backups before apply.

Project tree

File explorer limited to the folder you trust. Path traversal is blocked; the AI only sees what you attach or what you send in a request.

Rules & skills

Discovers AGENTS.md, CLAUDE.md, .cursorrules, .cursor/rules, and more — toggle which rules join the system prompt.

AI panel & agent

Streaming chat with shortcuts and rich context; Agent mode runs a tool loop (files, git, shell, search, and more) with approvals. Streaming tool cards, tasks, memory, and sub-agents ship in-app — see Capabilities.

Context adder

Attach images, stripped web pages, files, selections, tree snapshots, and paste — including drag-and-drop onto the panel.

Terminal & tooling

Integrated xterm.js terminal; the agent uses run_shell and other tools under policies you set — not plain unapproved fenced blocks.

Privacy posture

No first-party telemetry. Optional Router Studio account: encrypted vault on disk for settings and keys (not a cloud login). Optional email verification when your distributor configures a verify server. See Privacy.

Get started in minutes

Router Studio talks to models through OpenRouter. You supply the API key; usage and billing follow OpenRouter’s plans — Router Studio does not charge per request.

  1. Create an OpenRouter account and generate an API key at openrouter.ai/keys (keys typically start with sk-or-).
  2. Install Router Studio for your platform, then open Settings and paste the key. Use Test API Key to verify.
  3. Choose Open project folder, pick a repo, enable the rules you want, then open the model picker (Ctrl/Cmd + Shift + M) and start a conversation.
  4. Optional: open File → Router Studio account… (or Account in the top bar) to create a local encrypted vault for your settings and API keys on this device. If your build enables email verification, you will receive a code before registration completes — the same flow as the interactive preview on this page.

Developers can also clone the repository and run npm install and npm run dev — see the README.
